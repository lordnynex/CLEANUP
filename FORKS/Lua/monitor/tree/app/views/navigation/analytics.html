You can use this Analytics view to manage Dashboards. A dashboard is made of one or more charts.

Each chart is a group of lines displaying one or more projections about properties of traces during a period of time. It's possible to filter and aggregate this data in various ways.

Some examples of graphs could be:

  • Show the average time of all the requests to the service
  • Display the same thing, but only for the requests that returned errors
  • Restrict the previous graph to the requests with a generic path like /users/{user_id}.xml
